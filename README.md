HR Analytics Dashboard

This project focuses on analyzing employee data to uncover insights related to attrition, performance, and workforce trends using Power BI.

Key Features
1.Interactive dashboard for employee analysis
2.Attrition rate and employee distribution insights
3.Department-wise and gender-wise analysis
4.Performance and job satisfaction tracking


Tools & Technologies
1.Power BI
2.Excel / CSV Dataset
3.Data Cleaning & Transformation



Key Insights
1.Identified factors affecting employee attrition
2.Analyzed trends across departments and roles
3.Highlighted patterns in employee performance and satisfaction



Outcome

Developed a user-friendly dashboard to help organizations make data-driven HR decisions and improve workforce management.
